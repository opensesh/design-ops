# Skills Directory

Production-ready skills organized by category.

## Categories

| Folder | Purpose | Examples |
|--------|---------|----------|
| `brand/` | Brand identity, voice, and guidelines | Brand voice, tone guidelines, identity review |
| `design/` | Visual design and UX patterns | Design feedback, accessibility review, component patterns |
| `client/` | Client relations and communication | Proposals, presentations, feedback handling |
| `operations/` | Business processes and workflows | Project management, documentation, reporting |
| `development/` | Technical implementation | Code review, frontend patterns, architecture |

## Adding a New Skill

1. Identify the most appropriate category
2. Copy `../templates/skill-template.md` as your starting point
3. Name the file descriptively: `category/skill-name.md`
4. Test the skill in Claude Desktop before committing
5. Update this README if adding a new category
