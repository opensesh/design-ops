# Brand Voice

Write content that matches a defined brand voice and tone.

## Purpose

This skill ensures all written content maintains consistent brand personality across communications, marketing materials, and client-facing documents.

## When to Activate

Use this skill when:
- Writing marketing copy or website content
- Drafting client communications
- Creating social media posts
- Reviewing content for brand consistency

## Instructions

### Core Behavior

Before writing any content, establish the brand voice parameters:
1. Ask for or reference existing brand voice guidelines
2. Identify the audience and context
3. Match tone to the appropriate register (formal/casual/technical)

### Voice Dimensions

Evaluate and apply these voice characteristics:

| Dimension | Spectrum |
|-----------|----------|
| Formality | Casual ←→ Formal |
| Energy | Calm ←→ Energetic |
| Humor | Serious ←→ Playful |
| Authority | Peer ←→ Expert |
| Warmth | Professional ←→ Friendly |

### Process

1. **Audit** - Review existing brand materials if available
2. **Define** - Establish voice parameters for the context
3. **Write** - Create content matching those parameters
4. **Verify** - Check against brand guidelines
5. **Refine** - Adjust for consistency

### Output Format

When creating branded content, include:
- The content itself
- Brief note on voice choices made
- Suggestions for variations if applicable

## Constraints

- Never use voice characteristics that contradict stated guidelines
- Avoid generic corporate speak unless specifically requested
- Don't assume brand voice without asking for context first

## Examples

### Example 1: Tech Startup (Casual, Energetic)

**Input:**
```
Write a product announcement for our new feature. Brand voice: casual, energetic, peer-level.
```

**Output:**
```
Big news, friends!

You asked, we built. Say hello to Smart Sync — your projects
now talk to each other automatically. No more copy-paste marathons.

We're rolling this out to everyone this week. Go try it and
let us know what you think.

[Voice note: Conversational tone, direct address, excitement
without hyperbole]
```

### Example 2: Professional Services (Formal, Authoritative)

**Input:**
```
Write a service description. Brand voice: formal, expert, warm.
```

**Output:**
```
Our strategic consulting practice partners with leadership teams
to navigate complex organizational challenges. We bring deep
industry expertise and a collaborative approach to every engagement.

With over two decades of experience across sectors, we understand
that lasting change requires both rigorous analysis and genuine
human connection.

[Voice note: Professional register, expertise signals, warmth
through "partnership" framing]
```
